# Waggin Wheels fall detection module

## Components

### ATTiny85 microcontroller

### ADXL345 Accelerometer

### Passive Buzzer

### Red LED

### White LED

### 100ohm resistor (x2)

## Wiring Connections

| Pin on ATtiny85 | Connected to       |
|-----------------|--------------------|
| Pin 2 (PB3)     | Buzzer+            |
| Pin 3 (PB4)     | White LED+         |
| Pin 4 (GND)     | Ground             |
| Pin 5 (PB0/SDA) | ADXL345 SDA        |
| Pin 6 (PB1)     | Red LED+           |
| Pin 7 (PB2)     | ADXL345 SCL        |
| Pin 8 (VCC)     | 3.3V Power Supply  |
